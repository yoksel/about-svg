Что такое SVG?

Scalable Vector Graphics — формат векторных изображений, основанный на XML.

  • Масштабируется
  • Четкий на ретине
  • Человекопонятный код
  • Можно написать руками
  • Исходник
  • Переиспользуемые компоненты
  • Внутри — CSS и JavaScript
  • Хорошая поддержка браузерами
  • Доступность

Фигуры

<path>

  • M, m — moveto
  • Z — closepath
  • L, l — lineto
  • H, h — horizontal lineto
  • V, v — vertical lineto
  • C, c — curveto
  • S, s — smooth curveto
  • Q, q — quadratic Bézier curveto
  • T, t — smooth quadratic Bézier curveto
  • A, a — elliptical arc
